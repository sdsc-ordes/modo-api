modos.helpers.parse_region(region=None)[source]#

Parses an input UCSC-format region string into (reference_name, start, end).

Examples

>>> parse_region('chr1:10-320')
('chr1', 10, 320)
>>> parse_region('chr-1ba:10-320')
('chr-1ba', 10, 320)
>>> parse_region('chr1:-320')
('chr1', None, 320)
>>> parse_region('chr1:10-')
('chr1', 10, None)
>>> parse_region('chr1:10')
('chr1', 10, None)
>>> parse_region('chr1')
('chr1', None, None)
>>> parse_region('*')
('*', None, None)
>>> parse_region('')
(None, None, None)
Parameters:

region (Optional[str])

Return type:

tuple[str, Optional[int], Optional[int]]
